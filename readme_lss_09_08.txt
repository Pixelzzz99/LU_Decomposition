АЛГОРИТМ:
Программа принимает на вход матрицу и вектор и выводит в выходной файл вектор, после применения метода LU разложения.
1) L[i][1] = A[i][1]
   U[1][k] = A[1][k] / L[1][1]
    
2) L[i][j] = A[i][j] - SUM(L[i][1] * U[1][j], j=1,k-1)
    U[j][k] = A[j][k] - SUM(L[j][1] * U[1][k]/L[i][i], j=1,k-1)

    где:
    L - матрица нижней треугольной производной
    U - матрица верхней треугольной производной
    A - исходная матрица
    k - количество строк в матрице
    i - номер строки
    j - номер столбца

3) Полученное решение X является решением системы AX = B
